Credit Card Apps
================


## Starting the locally

### You need to manually install:

  - [Node.js](http://nodejs.org/): Amazing javascript asynchronous IO library, install manually.
  - [MongoDB](http://www.mongodb.org): NoSQL Database, install manually.
  - [NPM](http://npmjs.org/): Node package manager, used to install the remaining.
  - [Grunt](http://gruntjs.com/) : Compiler Frontends Assets (less/css, js, images) If you haven't used before, be sure to check out the [Getting Started](http://gruntjs.com/getting-started) guide

```
git clone git@github.com:loangarage/boilerplate.git
cd boilerplate
composer install
npm install
grunt
```
